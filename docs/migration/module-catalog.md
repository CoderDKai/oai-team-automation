# 模块清单

## core

- 职责: 配置、日志、模型、迁移与流程编排
- 边界: 不直接包含外部服务调用细节
- 负责人: 待补充

## cli

- 职责: CLI 入口、参数解析与命令调度
- 边界: 不包含业务流程实现
- 负责人: 待补充

## automation

- 职责: 浏览器自动化与流程执行
- 边界: 仅处理自动化流程
- 负责人: 待补充

## team

- 职责: Team 邀请、账号状态与席位检查
- 边界: 仅处理 Team 相关 API
- 负责人: 待补充

## email

- 职责: 邮箱创建与验证码获取
- 边界: 不包含 Team 或授权逻辑
- 负责人: 待补充

## auth

- 职责: 授权服务对接 (CRS/CPA/S2A)
- 边界: 授权逻辑不与业务流程混杂
- 负责人: 待补充

## single

- 职责: 单人/批量脚本与历史流程
- 边界: 逐步迁移至核心工作流
- 负责人: 待补充
